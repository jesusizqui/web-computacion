HTML: Estructura y Diseño

¿Qué es HTML?

HTML significa HyperText Markup Language (Lenguaje de Marcado de Hipertexto). Es el lenguaje de marcado utilizado para estructurar contenido en la web. Fue creado por Tim Berners-Lee en 1991 y es la base de cualquier página web.

Diferencias entre HTML5 y HTML4

HTML5 es la última versión de HTML y ofrece varias mejoras y características nuevas en comparación con HTML4, como:

  • Nuevos elementos semánticos: <header>, <footer>, <main>, <section>, <article>, <figure>, <aside>, <nav>, entre otros.
  • Elementos multimedia: <audio>, <video> y <source>.
  • Elementos gráficos: <svg> y <canvas>.
  • Nuevos elementos de formularios: <input type="date">, <input type="time">, <datalist>, <output>.

Estructura Básica de una Página HTML

Una página HTML básica incluye los siguientes bloques:

  1. <!DOCTYPE html>: Declara el tipo de documento como HTML.
  2. <html>: Contenedor raíz que encierra todo el contenido HTML.
  3. <head>: Contiene los metadatos, como el título de la página y los enlaces a las hojas de estilo.
  4. <body>: Contiene todo el contenido visible en la página.

Ejemplo Básico de HTML

<!DOCTYPE html>
<html>
  <head>
    <meta charset="UTF-8">
    <title>Mi Página Web</title>
  </head>
  <body>
    <h1>Bienvenido a mi página web</h1>
    <p>¡Hola, mundo!</p>
  </body>
</html>
